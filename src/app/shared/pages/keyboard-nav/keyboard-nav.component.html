Keyboard Navigation

It's incredibly important users be able to navigate your website using the keyboard only. If your website is functional only by keyboard, it's more likely to also be functional with other assistive technologies.

You can ensure keyboard usability be using native elements, ensuring interactive elements are in the tab focus, and ensuring non-interactive elemnts are NOT in tab focus.

Custom Components

When using custom components, you will not be able to directly add ARIA attributes or other accessibility solutions to have it trickle down to the semantic elements below. Instead, you will need to ensure you are programmatically adding your ARIA attributes at the source.
